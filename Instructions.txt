Game is set up to run local host but will work with an IP address but for easy of use we've kept it local host. 

1.To start a multiplayer game, register two users or log in with two accounts.
2.Select multiplayer.
3.Enter the name of a game session.
4.To enter the same game use the same name. 
5.Hit ready and play. 


server scripts:
Network-handler listens for connections/recieves messages
GameHandler takes care of sessions
GameLogic keeps track of all players in a session/score/timer
Player class is a simple container class.
